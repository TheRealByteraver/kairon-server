Welcome to the API.

Available endpoint(s): /token, /token/:token_name

Available request methods for /token:

  • GET: get all tokens
    returns: an array of token objects

  • POST: add a new token. e.g. for body: { "id": "bitcoin" }
    returns: created token object on success, error object on error

Available request methods for /token/:token_name:

  • GET: get a specific token, e.g. /token/bitcoin
    returns: a token object on success, error object on error

  • PUT: update a specific token. e. g. /token/ethereum with body { "active": 1 }
    returns: the updated token object on success, error object on error

  • DELETE: (hard) delete a specific token. e. g. /token/ethereum
    returns: the deleted token object on success, error object on error
